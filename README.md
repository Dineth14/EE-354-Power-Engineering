## EE-354 Power Engineering 

This repository contains resources and code related to load flow analysis in power engineering, specifically for the EE-354 course. Load flow analysis is essential for understanding the steady-state operation of electrical power systems.

### Contents

The repository is organized as follows:

- **`src/`**: Contains Python scripts for various load flow analysis methods.
    - `Newton_Raphson_Method.py`
    - `Gauss_Seidel_Load_Flow.py`
    - `Fast_Decoupled_Method.py`
    - And more...
- **`notebooks/`**: Contains Jupyter notebooks for interactive analysis.
    - `Load_Flow_Analysis.ipynb`
- **`docs/`**: Documentation and course materials.
    - `ASSIGNMENT_GUIDE.md`: Detailed guide on using the scripts.
    - `course_materials/`: PDFs, PPTs, and other course-related documents.

### Getting Started

To get started with the load flow analysis code, clone the repository and install the required dependencies:

```bash
git clone https://github.com/yourusername/ee-354-load-flow-analysis.git
cd ee-354-load-flow-analysis
pip install -r requirements.txt
```

### Usage

You can run the load flow analysis scripts from the root directory. For example, to run the Newton-Raphson method comparison:

```bash
python src/Load_Flow_Comparison.py
```

For more detailed instructions, please refer to [docs/ASSIGNMENT_GUIDE.md](docs/ASSIGNMENT_GUIDE.md).
